## Prerequisites

Before you begin, ensure you have:

- A modern web browser (Chrome, Firefox, or Safari)
- An email address for account verification
- Optional: Existing Markdown files or GitHub repositories for import

<Callout kind="info">

LAURYN DONG is free to start. Upgrade to Pro for unlimited spaces and advanced features.

</Callout>

## Sign Up for an Account

Follow these steps to create your LAURYN DONG account.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Open your browser and navigate to `https://app.lauryn-dong.com`.
  </Step>
  <Step title="Enter Your Details" icon="user">
    Click **Sign Up** and provide your email, name, and a strong password.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification link from `no-reply@lauryn-dong.com` and click it.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Answer a few quick questions about your project needs to customize your dashboard.
  </Step>
</Steps>

## Create Your First Documentation Space

A documentation space organizes all pages for a single project.

<Steps>
  <Step title="Click New Space" icon="plus">
    From the dashboard, select **New Space**.
  </Step>
  <Step title="Name Your Space" icon="edit-3">
    Enter a name like `My Project Docs` and select a color (e.g., `#3B82F6` for blue).
  </Step>
  <Step title="Add Description" icon="file-text">
    Optionally, add a brief description: "Documentation for My Project v1.0".
  </Step>
  <Step title="Save and Enter" icon="arrow-right">
    Click **Create**. You'll land in your new space's editor.
  </Step>
</Steps>

## Import Existing Documents

Import content from various sources to populate your space quickly.

<Tabs>
  <Tab title="Markdown Files" icon="file">
    Drag and drop `.md` files into the space or use the import button.

````markdown
# Welcome to My Project

## Features

- Fast setup
- Real-time editing

## Quick Start

```javascript
console.log("Hello, LAURYN DONG!");
```
````
  </Tab>
  <Tab title="GitHub Repo" icon="github">
    Connect your GitHub account and select a repo. LAURYN DONG syncs Markdown files automatically.
  </Tab>
  <Tab title="Google Docs" icon="file-text">
    Paste a shareable Google Docs link. Content converts to MDX format.
  </Tab>
</Tabs>

<Callout kind="tip">

Use the `{filename}` placeholder in imports for dynamic titles.

</Callout>
