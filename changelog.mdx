<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix", "improvement"]}>

## New Features

- Added advanced full-text search across all documentation pages with fuzzy matching.
- Introduced version pinning for docs pages to lock specific releases.

## Improvements

- Enhanced performance for large documentation sites, reducing load times by `>40%`.
- Updated editor with real-time preview for MDX components.

## Bug Fixes

- Fixed rendering issues with nested code blocks in Steps and Tabs components.
- Resolved authentication errors when using `{API_KEY}` in custom integrations.

</Update>

## Upgrade Guide

Follow these steps to upgrade your LAURYN DONG instance to the latest version.

<Steps>
  <Step title="Check Current Version" icon="package">

    Run the version check command:

````bash
npm list lauryn-dong
````

  </Step>
  <Step title="Backup Your Docs" icon="database">

    Export your documentation:

    ```bash
    lauryn-dong export --output ./backup
    ```

  </Step>
  <Step title="Install Latest" icon="download">

    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install lauryn-dong@latest
    ````

    ````bash
    yarn add lauryn-dong@latest
    ````
    </CodeGroup>

  </Step>
  <Step title="Restart Server" icon="refresh-cw">

    Restart your documentation server:

    ```bash
    lauryn-dong serve
    ```

  </Step>
</Steps>

<Callout kind="tip">
  Test upgrades in a staging environment first. Review migration notes in each release for potential breaking changes.
</Callout>
