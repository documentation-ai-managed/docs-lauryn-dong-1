## Webhook Configuration

Set up webhooks to trigger external actions on events like document updates or page publishes. Navigate to your LAURYN DONG dashboard at `https://dashboard.example.com/settings/webhooks` and add a new webhook.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Enter your webhook URL, such as `https://your-webhook-url.com/webhook`, and select events like `doc.updated` or `page.published`.
  </Step>
  <Step title="Verify Secret" icon="shield">
    Generate a secret token in LAURYN DONG and include it in the `X-Webhook-Secret` header for verification.
  </Step>
  <Step title="Test Endpoint" icon="play">
    Send a test payload to confirm delivery.
  </Step>
</Steps>

Here is an example webhook payload:

<CodeGroup tabs="JSON">
  ```json
  {
    "event": "doc.updated",
    "doc_id": "doc_123abc",
    "timestamp": "2024-10-15T12:00:00Z",
    "data": {
      "title": "API Reference",
      "url": "https://docs.example.com/api"
    }
  }
  ```
</CodeGroup>

<Callout kind="tip">
  Always validate the `X-Webhook-Secret` header on your server to prevent unauthorized requests.
</Callout>

## API Endpoints for Custom Integrations

Use LAURYN DONG's REST API at `https://api.example.com/v1` to create, update, or fetch documentation programmatically.

<ParamField path="docs" param-type="GET" required="true">
  Retrieve a list of your documents.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer `{YOUR_API_KEY}` for authentication.
</ParamField>

Example request:

<Request tabs="JavaScript,cURL">
  ````javascript
  const response = await fetch('https://api.example.com/v1/docs', {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  });
  const docs = await response.json();
  console.log(docs);
  ````

  ````bash
  curl -H "Authorization: Bearer YOUR_API_KEY" \
       https://api.example.com/v1/docs
  ````
</Request>

<Response tabs="200">
  ```json
  {
    "docs": [
      {
        "id": "doc_123abc",
        "title": "Getting Started",
        "updated_at": "2024-10-15T12:00:00Z"
      }
    ]
  }
  ```
</Response>

## Export Options

Export your documentation to various formats for sharing or migration.

<Tabs>
  <Tab title="Markdown" icon="file-text">
    Download entire sites or single pages as Markdown files. Ideal for GitHub repos.
    
    ```
    https://dashboard.example.com/export/markdown?doc_id=doc_123abc
    ```
  </Tab>
  <Tab title="PDF" icon="file-text">
    Generate printable PDFs with custom branding.
    
    ```
    https://dashboard.example.com/export/pdf?include_toc=true
    ```
  </Tab>
  <Tab title="HTML" icon="code">
    Export static HTML for hosting on any web server.
    
    <Callout kind="info">
      HTML exports include all images and styles.
    </Callout>
  </Tab>
</Tabs>
