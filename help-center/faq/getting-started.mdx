## Overview

To begin using LAURYN DONG, create your first documentation project.

## Steps

1. Sign in at `https://docs.lauryndong.com`  
2. Click "New Project" on your dashboard
3. Enter a name and description
4. Start adding documentation pages

## Next

Explore features and invite team members from the Project Settings.
