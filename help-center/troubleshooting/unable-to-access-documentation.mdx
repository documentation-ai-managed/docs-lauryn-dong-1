---
title: Unable to Access Documentation
description: Quick fix if you can't view docs.
---

## Symptoms

- Docs page won't load
- Access denied error

## Causes

- Not signed in
- No project permissions

## Solution

<Steps>
  <Step title="Check if signed in">
    ````
    Go to https://docs.lauryndong.com and log in.
    ````
  </Step>
  <Step title="Confirm permissions">
    Ask a project admin to add you to the project if you lack access.
  </Step>
</Steps>
